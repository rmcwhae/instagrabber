A simple plugin that fetches the latest posts from a specified Instagram account.

== Description ==

This simple plugin displays the latest posts of a specified Instagram account. Once activated, an admin settings page is available in `Admin > Settings > Instagrabber`. It can then be called by the `[instagrabber]` shortcode in posts and pages.

*NOTE:* This plugin was created for a coding test and is not necessarily intended to be production-quality code.

== Installation ==

1. Unzip `russell-instagram-fetcher.zip` and upload it to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Specify an Instagram username and post limit in `Admin > Settings > Instagrabber`. The default post limit is 5.
1. Place the `[instagrabber]` shortcode in your page or posts.
1. Styling can be changed by modifying `russell-instagram-fetcher\public\css\russell-instagram-fetcher-public.css`.
